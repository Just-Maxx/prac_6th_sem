# Метод продолжения по параметру для краевых задач ОДУ

Проект по практикуму: реализация метода продолжения по параметру для решения краевых задач обыкновенных дифференциальных уравнений.

## Идея метода

Краевая задача вида

dx/dt = f(t, x),  
R(x(a), x(b)) = 0

сводится к нелинейному уравнению

Phi(p) = 0,

где p — параметр задачи Коши.

На текущем этапе реализована первая часть:
- решение внутренней задачи Коши;
- вычисление граничной невязки Phi(p);
- тестовый пример с гармоническим осциллятором.

## Структура проекта

- `main.py` — запуск тестового примера;
- `src/models.py` — описание структуры краевой задачи;
- `src/examples.py` — тестовые задачи;
- `src/inner_solver.py` — решение внутренней задачи Коши;
- `src/residuals.py` — вычисление невязки граничных условий;
