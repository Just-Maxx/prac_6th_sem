# История изменений

## [0.2] - 30-04-2026

### Добавлено
- Добавлено вычисление якобиана правой части системы.
- Добавлено вычисление якобиана граничной невязки.
- Реализовано решение внутренней задачи вместе с уравнением в вариациях.
- Добавлено вычисление якобиана `Phi'(p)` через матрицу вариаций.
- Добавлена проверка якобиана методом конечных разностей.
- Обновлён вывод программы: теперь отображаются невязка, якобиан и разность якобианов.

### Изменено
- Расширен класс `BVPProblem`: добавлены поля `rhs_jacobian` и `boundary_jacobian`.
- Тестовый пример дополнен аналитическими производными.


## [0.1] - 27-04-2026

### Добавлено
- Создана начальная структура проекта.
- Добавлен класс `BVPProblem` для хранения постановки краевой задачи.
- Добавлен тестовый пример с гармоническим осциллятором.
- Реализовано решение внутренней задачи Коши.
- Реализовано вычисление граничной невязки `Phi(p)`.
- Добавлен базовый тест для проверки невязки на точном решении.
